## 🚀 Как использовать систему переводов

### Переключение языка в браузере
```
/locale/ru  - Переключиться на русский
/locale/en  - Переключиться на английский
/locale/tg  - Переключиться на таджикский
```

### В коде Blade
```blade
{{ __('ui.app_name') }}        <!-- "LegalAI Pro" -->
{{ __('ui.language') }}        <!-- "Язык", "Language", "Забон" -->
{{ __('ui.save') }}            <!-- "Сохранить", "Save", "Захира кардан" -->
```

### В PHP коде
```php
$text = __('ui.contract_analysis_title');  // Получить переведённый текст
app()->setLocale('tg');                    // Установить локаль
```
