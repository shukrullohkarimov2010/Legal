## 🌍 Как использовать переводчик

### Способ 1: Переключилось в браузере
Просто перейдите по одному из этих URL:
- `/locale/ru` - Русский
- `/locale/en` - English
- `/locale/tg` - Таҷикї

### Способ 2: В коде Blade шаблонов
```blade
{{ __('ui.app_name') }}           <!-- "LegalAI Pro" на всех языках -->
{{ __('ui.language') }}           <!-- "Язык" / "Language" / "Забон" -->
{{ __('ui.save') }}               <!-- "Сохранить" / "Save" / "Захира кардан" -->
{{ __('ui.main_navigation') }}    <!-- "Основная навигация" / "Main navigation" / "Паймони асосӣ" -->
```
